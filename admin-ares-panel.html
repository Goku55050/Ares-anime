Anime Data Editor (JSON)

This panel shows your current animeData as JSON. You can edit/add anime, seasons, episodes, streaming links, download links here.
When finished:

  1. Click Download JSON to save anime-data.json.
  2. Open that file on your PC/phone and copy JSON.
  3. Paste into script.js replacing existing animeData value.
  4. Commit + push to GitHub → Vercel will auto-deploy, and users will see updates.

Note: This admin panel does not directly change your live site. You must download JSON and update script.js in your repo.
